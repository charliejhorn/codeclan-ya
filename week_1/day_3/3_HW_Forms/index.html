Your Task

You are a freelancer software developer and you need to create a form to collect information from potential clients who would like to request a quote for some work. In order to help you judge the complexity and time requirements, and give an accurate quote, you have set a series of questions to assess the work.

  • Create a new folder for your homework called 'Forms Homework'.
  • Inside the 'Forms Homework' folder create two files:
    • index.html
    • style.css
  • Create a form that contains the following:
    • Three Input fields for:
      1. Full Name (Should accept text)
      2. Email (Should accept an email)
      3. Phone Number (Should accept a number and not text)
    • A small list of job-types that allows the user to select one option (e.g. Web Site / Mobile App / UX consultancy).
    • A checklist of features they are looking for where the user can select more than one (e.g. Home Page, Contact Form, Booking System, Shopping Cart, Form, Chatbot)
    • A submit Button

Additional Exercises

  • A dropdown menu that allows the user to select their location from a selection of options (Include 3 or 4 options, such as Scotland/England/Wales/Ireland)
  • A text area for the client to describe the work to be done.
  • Add styling of your choice to your form.

Hints

We have covered some basic form elements in class, however, you will need to do some further research into elements we have not covered, as well as the other types that can be used with the elements we discussed earlier.

What other types can an input have? How can we incorporate some of them into relevant fields?
